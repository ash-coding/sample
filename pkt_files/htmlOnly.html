Progateとは

Progateは、Web上で完結する初心者用プログラミング学習サイトです。環境構築が不要なことから、これまで多くの初学者の支持を得ています。

HTML・CSS・jQueryとは

HTML

HTMLは、Webページを作成するための最も基本的な言語で、インターネット上のWebページのほとんどにHTMLが使われています。

一般に、Webページで見られるテキストやリンク、画像などはHTMLで記述されています。逆に、文字の装飾や動的なアニメーションなどはHTMLでは表現できません。

ためしに、HTMLのみで作成したこちらのページをクリックしてみてください。味気ないページになっていると思います。

CSS

CSSは、Webページのデザインやレイアウトなどの見栄えを整えるための言語です。

例えば、このような文字の装飾とかはCSSですね！

jQuery

jQueryは、JavaScriptのライブラリの一つで、Webページに効果やアニメーションを実装できる言語です。

このページで言うと、目次の表示/非表示の切り替えやヘッダー右側のボタンで小窓を表示させるなどの動作が該当します。

実際にjQueryの効果を体験してもらうために、ボタンを2つ用意しました。2つのボタンは、どちらもページトップへのジャンプボタンです。 ただし、左側はHTMLでリンク先指定したもの、右側はjQueryで実装したものです。ぜひ違いを体感してください。
